리스트

순서 없는 리스트

  • 순서 없는 리스트를 쓰기 위해서는 ul 태그로 시작
  • 리스트의 요소들은 li로 채워줌
  • ul은 Unordered List

순서 있는 리스트

  1. 순서 있는 리스트는 ol
  2. 순서 없는 리스트와 사용법은 동일
  3. ol은 ordered list
  4. 순서 있는 리스트는 자동으로 숫자를 매겨줌

멀티미디어 삽입

이미지 삽입

  • 이미지 삽입을 위해서는 img 태그를 사용
  • 꼭 src 속성을 넣어주고, 값을 이미지의 주소로
  • alt 속성은 이미지의 설명이자, 찾는 이미지가 없을 때 보여줄 글씨

영상 삽입

  • 동영상 삽입은 video 태그
  • 사용법은 vidio 태그 요소 안에 source 태그로 이미지 주소와 type 넣어주기
  • 유튜브 영상도 삽입 가능

링크 삽입

  • 링크 삽입은 a tag
  • 사용법은 href 속성에 html의 주소를 넣어주기
  • href 속성에 도메인 주소도 입력 가능
